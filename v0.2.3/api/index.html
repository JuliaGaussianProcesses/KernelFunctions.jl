Kernel Functions

KernelFunctions.SqExponentialKernel — Type

SqExponentialKernel([ρ=1.0])

The squared exponential kernel is an isotropic Mercer kernel given by the formula:

    κ(x,y) = exp(-ρ²‖x-y‖²)

See also ExponentialKernel for a related form of the kernel or GammaExponentialKernel for a generalization.

KernelFunctions.GammaExponentialKernel — Type

GammaExponentialKernel([ρ=1.0,[gamma=2.0]]) The γ-exponential kernel is an isotropic Mercer kernel given by the formula:

    κ(x,y) = exp(-ρ^(2γ)‖x-y‖^(2γ))

KernelFunctions.ExponentiatedKernel — Type

KernelFunctions.MaternKernel — Type

MaternKernel([ρ=1.0,[ν=1.0]]) The matern kernel is an isotropic Mercer kernel given by the formula:

    κ(x,y) = 2^{1-ν}/Γ(ν)*(√(2ν)‖x-y‖)^ν K_ν(√(2ν)‖x-y‖)

For ν=n+1/2, n=0,1,2,... it can be simplified and you should instead use ExponentialKernel for n=0, Matern32Kernel, for n=1, Matern52Kernel for n=2 and SqExponentialKernel for n=∞.

source
KernelFunctions.Matern32Kernel — Type

Matern32Kernel([ρ=1.0]) The matern 3/2 kernel is an isotropic Mercer kernel given by the formula:

    κ(x,y) = (1+√(3)ρ‖x-y‖)exp(-√(3)ρ‖x-y‖)
source
KernelFunctions.Matern52Kernel — Type

Matern52Kernel([ρ=1.0]) The matern 5/2 kernel is an isotropic Mercer kernel given by the formula:

    κ(x,y) = (1+√(5)ρ‖x-y‖ + 5ρ²‖x-y‖^2/3)exp(-√(5)ρ‖x-y‖)

KernelFunctions.LinearKernel — Type

LinearKernel([ρ=1.0,[c=0.0]]) The linear kernel is a Mercer kernel given by

    κ(x,y) = ρ²xᵀy + c

Where c is a real number

source
KernelFunctions.PolynomialKernel — Type

PolynomialKernel([ρ=1.0[,d=2.0[,c=0.0]]]) The polynomial kernel is a Mercer kernel given by

    κ(x,y) = (ρ²xᵀy + c)^d

Where c is a real number, and d is a shape parameter bigger than 1

source
KernelFunctions.RationalQuadraticKernel — Type

RationalQuadraticKernel([ρ=1.0[,α=2.0]]) The rational-quadratic kernel is an isotropic Mercer kernel given by the formula:

    κ(x,y)=(1+ρ²||x−y||²/α)^(-α)

where α is a shape parameter of the Euclidean distance. Check GammaRationalQuadraticKernel for a generalization.

source
KernelFunctions.GammaRationalQuadraticKernel — Type

GammaRationalQuadraticKernel([ρ=1.0[,α=2.0[,γ=2.0]]]) The Gamma-rational-quadratic kernel is an isotropic Mercer kernel given by the formula:

    κ(x,y)=(1+ρ^(2γ)||x−y||^(2γ)/α)^(-α)

where α is a shape parameter of the Euclidean distance and γ is another shape parameter.

source

Kernel Combinations

KernelFunctions.KernelSum — Type

KernelSum(kernels::Array{Kernel};weights::Array{Real}=ones(length(kernels))) Create a positive weighted sum of kernels. One can also use the operator +

k1 = SqExponentialKernel()
k2 = LinearKernel()
k = KernelSum([k1,k2])
kernelmatrix(k,X) == kernelmatrix(k1,X).+kernelmatrix(k2,X)
kernelmatrix(k,X) == kernelmatrix(k1+k2,X)
kweighted = 0.5*k1 + 2.0*k2
source
KernelFunctions.KernelProduct — Type

KernelProduct(kernels::Array{Kernel}) Create a multiplication of kernels. One can also use the operator *

k1 = SqExponentialKernel()
k2 = LinearKernel()
k = KernelProduct([k1,k2])
kernelmatrix(k,X) == kernelmatrix(k1,X).*kernelmatrix(k2,X)
kernelmatrix(k,X) == kernelmatrix(k1*k2,X)

KernelFunctions.FunctionTransform — Type

FunctionTransform

    f(x) = abs.(x)
    tr = FunctionTransform(f)

Take a function f as an argument which is going to act on each vector individually. Make sure that f is supposed to act on a vector by eventually using broadcasting For example f(x)=sin(x) -> f(x)=sin.(x)

KernelFunctions.ChainTransform — Type

Chain a series of transform, here t1 will be called first

    t1 = ScaleTransform()
    t2 = LowRankTransform(rand(3,4))
    ct = ChainTransform([t1,t2]) #t1 will be called first
    ct == t2∘t1
