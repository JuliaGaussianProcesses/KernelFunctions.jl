Base Kernels

These are the basic kernels without any transformation of the data. They are the building blocks of KernelFunctions

Constant Kernels

Constant Kernel

The ConstantKernel is defined as

\[ k(x,x';c) = c,\]

where $c \in \mathbb{R}$.

White Kernel

The WhiteKernel is defined as

\[ k(x,x') = \delta(x-x').\]

Zero Kernel

The ZeroKernel is defined as

\[ k(x,x') = 0.\]

Cosine Kernel

The CosineKernel is defined as

\[ k(x, x') = \cos(\pi |x-x'|),\]

where $x\in\mathbb{R}$.

Exponential Kernels

Exponential Kernel

The ExponentialKernel is defined as

\[ k(x,x') = \exp\left(-|x-x'|\right).\]

Square Exponential Kernel

The SqExponentialKernel is defined as

\[ k(x,x') = \exp\left(-\|x-x'\|^2\right).\]

Gamma Exponential Kernel

The GammaExponentialKernel is defined as

\[ k(x,x';\gamma) = \exp\left(-\|x-x'\|^{2\gamma}\right),\]

where $\gamma > 0$.

Exponentiated Kernel

The ExponentiatedKernel is defined as

\[ k(x,x') = \exp\left(\langle x,x'\rangle).\]

Fractional Brownian Motion

The FBMKernel is defined as

\[ k(x,x';h) = \frac{|x|^{2h} + |x'|^{2h} - |x-x'|^{2h}}{2},\]

where $h$ is the Hurst index and $0<h<1$.

Gabor Kernel

The GaborKernel is defined as

\[ k(x,x'; l,p) =& h(x-x';l,p)\\ h(u;l,p) =& \exp\left(-\cos\left(\pi \sum_i \frac{u_i}{p_i}\right)\sum_i \frac{u_i^2}{l_i^2}\right),\]

where $l_i >0 $ is the lengthscale and $p_i>0$ is the period.

Matern Kernels

Matern Kernel

The MaternKernel is defined as

\[ k(x,x';\nu) = \frac{2^{1-\nu}}{\Gamma(\nu)}\left(\sqrt{2\nu}|x-x'|\right)K_\nu\left(\sqrt{2\nu}|x-x'|\right),\]

where $\nu > 0$.

Matern 3/2 Kernel

The Matern32Kernel is defined as

\[ k(x,x') = \left(1+\sqrt{3}|x-x'|\right)\exp\left(\sqrt{3}|x-x'|\right).\]

Matern 5/2 Kernel

The Matern52Kernel is defined as

\[ k(x,x') = \left(1+\sqrt{5}|x-x'|+\frac{5}{2}\|x-x'\|^2\right)\exp\left(\sqrt{5}|x-x'|\right).\]

Neural Network Kernel

The NeuralNetworkKernel (as in the kernel for an infinitely wide neural network interpretated as a Gaussian process) is defined as

\[ k(x, x') = \arcsin\left(\frac{\langle x, x'\rangle}{\sqrt{(1+\langle x, x\rangle)(1+\langle x',x'\rangle)}}\right).\]

Periodic Kernel

The PeriodicKernel is defined as

\[ k(x,x';r) = \exp\left(-0.5 \sum_i (sin (π(x_i - x'_i))/r_i)^2\right),\]

where $r$ has the same dimension as $x$ and $r_i >0$.

Piecewise Polynomial Kernel

The PiecewisePolynomialKernel is defined as

\[ k(x,x'; P, V) =& \max(1 - r, 0)^{j + V} f(r, j),\\ r =& x^\top P x',\\ j =& \lfloor \frac{D}{2}\rfloor + V + 1,\]

where $x\in \mathbb{R}^D$, $V \in \{0,1,2,3\} and $P$ is a positive definite matrix. $f$ is a piecewise polynomial (see source code).

Polynomial Kernels

Linear Kernel

The LinearKernel is defined as

\[ k(x,x';c) = \langle x,x'\rangle + c,\]

where $c \in \mathbb{R}$

Polynomial Kernel

The PolynomialKernel is defined as

\[ k(x,x';c,d) = \left(\langle x,x'\rangle + c\right)^d,\]

where $c \in \mathbb{R}$ and $d>0$

Rational Quadratic

Rational Quadratic Kernel

The RationalQuadraticKernel is defined as

\[ k(x,x';\alpha) = \left(1+\frac{\|x-x'\|^2}{\alpha}\right)^{-\alpha},\]

where $\alpha > 0$.

Gamma Rational Quadratic Kernel

The GammaRationalQuadraticKernel is defined as

\[ k(x,x';\alpha,\gamma) = \left(1+\frac{\|x-x'\|^{2\gamma}}{\alpha}\right)^{-\alpha},\]

where $\alpha > 0$ and $\gamma > 0$.

Spectral Mixture Kernel

The spectral mixture kernel is called by spectral_mixture_kernel.

Wiener Kernel

The WienerKernel is defined as

\[k(x,x';i) = \left\{\begin{array}{cc} \delta(x, x') & i = -1\\ \min(x,x') & i = 0\\ \frac{\min(x,x')^{2i+1}}{a_i} + b_i \min(x,x')^{i+1}|x-x'|r_i(x,x') & i\geq 1 \end{array}\right.,\]

where $i\in\{-1,0,1,2,3\}$ and coefficients $a_i$, $b_i$ are fixed and residuals $r_i$ are defined in the code.

Composite Kernels

Transformed Kernel

The TransformedKernel is a kernel where input are transformed via a function f

\[ k(x,x';f,\widetile{k}) = \widetilde{k}(f(x),f(x')),\]

Where $\widetilde{k}$ is another kernel and $f$ is an arbitrary mapping.

Scaled Kernel

The ScaledKernel is defined as

\[ k(x,x';\sigma^2,\widetilde{k}) = \sigma^2\widetilde{k}(x,x')\]

Where $\widetilde{k}$ is another kernel and $\sigma^2 > 0$.

Kernel Sum

The KernelSum is defined as a sum of kernels

\[ k(x, x'; \{k_i\}) = \sum_i k_i(x, x').\]

KernelProduct

The KernelProduct is defined as a product of kernels

\[ k(x,x';\{k_i\}) = \prod_i k_i(x,x').\]

Tensor Product

The TensorProduct is defined as :

\[ k(x,x';\{k_i\}) = \prod_i k_i(x_i,x'_i)\]
